# Магазин ароматических свечей

## Описание программы

«Магазин ароматических свечей» — это настольное приложение для автоматизации процесса продажи свечей. Программа позволяет просматривать каталог товаров, добавлять свечи в корзину, изменять количество и оформлять заказы с сохранением в базе данных.

## Требования к системе

- Операционная система: Windows 10 или Windows 11
- Установленный .NET 9 Runtime
- SQL Server (достаточно LocalDB или SQL Server Express)
- 2 ГБ оперативной памяти

## Установка и запуск

1. Скачайте и распакуйте архив с программой.
2. Откройте решение CandleStore.sln в среде Rider или Visual Studio.
3. Восстановите NuGet-пакеты (команда `dotnet restore`).
4. Создайте базу данных (команда `dotnet ef database update`).
5. Запустите программу (клавиша F5 или кнопка Run).

## Как пользоваться программой

### Главное окно

Главное окно разделено на две части. Слева отображается список всех свечей, доступных для покупки. Справа — детальная информация о выбранной свече и корзина для покупок.

### Выбор товара

Кликните по названию свечи в левом списке. Справа появятся подробные сведения: название, аромат, цена и количество на складе.

### Добавление в корзину

1. Выберите свечу из списка слева.
2. При необходимости измените количество с помощью кнопок «+» и «-».
3. Нажмите кнопку «Добавить в корзину».

Товар появится в списке корзины, а общая сумма заказа пересчитается автоматически.

### Оформление заказа

1. Убедитесь, что в корзине есть хотя бы один товар.
2. Нажмите кнопку «Оформить заказ».
3. В появившемся окне введите ваше имя.
4. Нажмите «ОК».

После успешного оформления заказа вы увидите сообщение с номером заказа. Корзина очистится, а количество товаров на складе уменьшится.

### Выход из программы

Закройте главное окно программы стандартным способом (крестик в правом верхнем углу).

## Возможные ошибки и их решение

| Сообщение об ошибке | Причина | Решение |
|---------------------|---------|---------|
| Выберите товар из списка | Не выбран товар перед добавлением в корзину | Выберите свечу из левого списка |
| Введите корректное количество | Указано ноль или отрицательное число | Введите число от 1 и выше |
| Недостаточно товара на складе | Запрошено больше, чем есть в наличии | Уменьшите количество |
| Корзина пуста | Заказ оформляется без товаров | Добавьте хотя бы один товар в корзину |
| Имя покупателя не может быть пустым | Не введено имя при оформлении заказа | Введите ваше имя |
